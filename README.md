# 📚 PUC Exercises

Repositório destinado ao versionamento das atividades, exercícios e práticas desenvolvidas durante a graduação em **Sistemas de Informação – PUC Minas**.

---

## 🎯 Objetivos

- Centralizar exercícios acadêmicos em um único repositório
- Aplicar boas práticas de versionamento com Git
- Evoluir qualidade, organização e performance do código
- Criar base de revisão para provas e trabalhos

---

## 🧠 Conteúdos Abordados

Os exercícios presentes neste repositório podem envolver:

- Lógica de programação
- Estruturas condicionais e de repetição
- Métodos e modularização
- Recursividade
- Vetores e matrizes
- Estruturas de dados
- Algoritmos de ordenação
- Análise de complexidade
- Programação Orientada a Objetos (POO)
- Manipulação de arquivos
- Outros tópicos abordados nas disciplinas

---

## 🛠 Disciplinas do eixo de programação

- Algoritmos e Técnicas de Programação (ATP)
- Algoritmos e Estrutura de Dados (AED)
- Programação Orientada a Objetos (POO)
